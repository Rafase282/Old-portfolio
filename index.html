Portfolio

Here are some of my most recent Projects!

From older to newer

Random Quote Generator

This site will display random quotes using the Forismatic API. You will then have the option to tweet it and share with others. You might need to edit the post before submmiting as some quotes will go over the limit.

Technologies used: HTML5, CSS3, Boostrap3, jQuery2

Local Weather App

This site will get your location from your IP address using the IP-API.com API. The use that information to get the current weather and other basic information using the OpenWeatherMap API. The user has the option to display the weather and wind speed using the Metric or Imperial system. It also changes background image depending on the temperature.

Technologies used: HTML5, CSS3, Boostrap3, jQuery2

Pomodoro Timer

This web app is great for productivity, it can sit in any corner of your screen, and there is not even the need to keep it in view. Once you set the amount of time you want to work and how long you want to rest for, then just let it run and it will keep looking until you stop it. You are also able to pause it in case you have to step aside from your work for a brief moment. It has a nice progression animation to let you know how much time you have left and the break and work progress animation have different color to help you figure out in which one you should be if needed.

Technologies used: HTML5, CSS3, Boostrap3, jQuery2, Canvas

Twitchtv JSON API

This app will use the Twitch TV JSON API and display a list of accounts, if they are closed it will display the right symbols and mention that the account has been closed, the same for offline and online streams. It will also display online and offline streams on a different tab and has a global search function that display results as you type while hiding anything that is not part of the match using regular expressions.

Technologies used: HTML5, CSS3, Boostrap3, jQuery2, API, Font Awesome.

FCC Camper News

This web app displays the hottes news from FreeCodeCamp news. You have will see the username, the date it was posted, how many upvotes and the original link to the article.

Technologies used: HTML5, CSS3, Boostrap3, jQuery2, Datejs

Calculator

This calculator will allow you to do chain operations and get a proper answer thanks to the use of parentheses for more complex operations. It does cover the basic features of a calculator too.

Technologies used: HTML5, CSS3, Boostrap3, jQuery2

Wikipedia Searcher

This app will use the wikipedia API to produce search on the fly on their database. You can also click a button to get a random wiki article. The app can easily be adjusted for any other MediaWiki installation.

Technologies used: HTML5, CSS3, Boostrap3, jQuery2

Simon Game

Web App that lets the use plays against the machine. There are two modes, strict mode which does not allow for reties, and normal mode which allows for unlimited retries.
The speed increases after the 5th,9th and 13th round. The player wins after the 20th round.

Technologies used: HTML5, CSS3, Boostrap3, jQuery2, Jade, SASS.

Timestamp Microservice API

This is a microservice that will allow get calls to convert Unix time-stamps or natural dates to one or the other. It checks the date given for validity and output either null or a predefined date format.

Technologies used: HTML5, Boostrap3, Font Awesome, Javascript, Momentjs, Nodejs, Expressjs, JSON
